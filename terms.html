Documentation Log'In App

Avez-vous des interrogations sur l'utilisation de l'application ou des difficultés à comprendre la présentation des données de votre logette ?

Nous vous proposons un guide d'utilisation détaillé ci-dessous.

Comment avoir la kimbox et un compte pour se connecter à l'application ?

Pour se proccurer une logette, veiller contacter Kim Engineering au agress@kim.com et remplir le formulaire d'ennregistrement, après traitement des données, une logette vous sera attribuer ainsi qu'un compte utilisateur pour avoir l'accès à l'application pour le monitoring de sa logette.


Comment télécharger l'application Log'in App de la kimbox?

Pour télécharger l’application Log'In App, suivez ces étapes simples :

Accédez à la plateforme de téléchargement :
- Pour les utilisateurs d'iPhone, ouvrez l'App Store.
- Pour les utilisateurs d'Android, ouvrez Google Play.
Recherchez l'application dans la barre de recherche, tapez "Log'In App".
Téléchargez et installez :
Cliquez sur le bouton Télécharger ou Installer. Attendez que le téléchargement soit terminé.
Ouvrez l'application une fois l'installation terminée.


Comment se connecter dans son application mobile ?

Pour accéder à l'application mobile, vous devez saisir votre adresse e-mail et votre mot de passe si vous possédez déjà un compte (en tant que client). Sinon, veuillez vous rendre dans nos locaux pour vous enregistrer.


Comment allumer et éteindre votre logette à l'aide de l'application mobile?


Pour allumer votre logette : connectez-vous à votre application. Une fois connecté, cliquez sur le bouton Paramètres sur la page d'accueil.

Dans la page des paramètres, vérifiez si le bouton d'alimentation est désactivé (il doit être en gris). Si c'est le cas, appuyez dessus pour l'activer. Par contre, s'il est en vert, cela signifie que votre logette est déjà allumée.


Pour éteindre votre logette : connectez-vous à votre application. Une fois connecté, cliquez sur le bouton Paramètres sur la page d'accueil.

Dans la page des paramètres, vérifiez si le bouton d'alimentation est activé (il doit être en vert). Si c'est le cas, appuyez dessus pour le désactiver. Sinon si le bouton est en gris, cela signifie que votre logette est déjà éteinte.


Comment voir les données de sa logette qui arrivent en temps réel ?

Pour accéder aux données en temps réel de votre kimbox, connectez-vous à votre application mobile. Une fois connecté, cliquez sur le bouton Paramètres sur la page d'accueil.

Dans la page des paramètres, vous pourrez visualiser la température, l'humidité et la puissance de votre kimbox, ainsi que les valeurs de tension pour vos trois lignes.




Comment voir la consommation de votre maison ?

Pour voir la consommation de votre maison,
connectez-vous dans votre application. Une fois connecté, dans la page d’accueil cliquez sur le bouton Consommation.

Dans la page de consommation, vous verrez les graphiques et les tableaux représentant votre consommation journalière (5 derniers jours), et votre consommation mensuelle (5 derniers mois).
Appuyez sur le button + de chaque card pour voir le graphique ou le tableau.


Comment savoir si votre logette est en surchauffe ?

Si la température de votre logette commence à augmenter et dépasse le niveau normal, l'application vous enverra une notification pour vous alerter à ce sujet.

Et pour voir cette notification, il vous suffit d'ouvrir l'application et cliquer sur le bouton des notifications en haut à droite dans la page d'accueil.
